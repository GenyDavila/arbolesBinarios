# arbolesBinarios
Programa sencillo que permite crear árboles binarios (no balanceados) y realizar algunas operaciones básicas:
  1. Insertar un nodo
  2. Buscar un nodo
  3. Recorrer el árbol (inorden, preorden, postorden)
  
Al compilar y ejecutar el código el programa muestra el siguiente menú de inicio:
  
                        === ARBOLES BINARIOS ===
                        
        Insertar nodo.........................................[1]
        Mostrar recorridos (inorden, preorden, postorden).....[2]
        Buscar nodo...........................................[3]
        Salir.................................................[4]
        
        Seleccione una opcion:
        
 NOTA: El programa no realiza la validación de tipos de datos, si los datos que se ingresan son distintos a números enteros el programa dejará de funcionar.
